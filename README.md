# computer-networkThis project implements a Virtual Local Area Network (VLAN) using Cisco Packet Tracer with two different network topologies. VLAN 10 is configured using a Star Topology, where all end devices are connected through a central switch, providing simple management and reliable communication. VLAN 20 is configured using a Mesh Topology, where devices have multiple interconnected paths to improve reliability and fault tolerance. The project demonstrates VLAN creation, port assignment, and logical network segmentation. Devices within the same VLAN can communicate successfully, while communication between different VLANs is isolated unless inter-VLAN routing is enabled. This simulation helps understand VLAN configuration, network security, traffic management, and the practical use of different topologies in modern computer networks.
